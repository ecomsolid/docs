#### Margin
>[!note]
>- Cách gọi tên: className = **.gt_v3_{m|mt|mr|mb|ml|mx|my}--{name}}**
>- Ex: gt_v3_mt--2, gt_v3_m--2...

Chú thích:
- m : margin
- mt : margin top
- mb : margin bottom
- ml : margin left
- mr : margin right
- mx : margin left + margin right
- my : margin top + margin bottom

>[!warning]
>Support Responsive {lg|md|sm|xs}:{className}
>Ex: md:gt_v3_mt--2
#### Padding
>[!note]
>- Cách gọi tên: className = **.gt_v3_{p|pt|pr|pb|pl|px|py}--{name}}**
>- Ex: gt_v3_mt--2, gt_v3_m--2...

Chú thích:
- p : padding
- pt : padding top
- pb : padding bottom
- pl : padding left
- pr : padding right
- px : padding left + padding right
- py : padding top + padding bottom

>[!warning]
>Support Responsive {lg|md|sm|xs}:{className}
>Ex: md:gt_v3_pt--2

### **Typography**
>[!note]
>Cách gọi tên: className = **.gt_v3_{typoName}**

Danh sách class:
- gt_v3_heading-1
- gt_v3_heading-2
- gt_v3_heading-3
- gt_v3_heading-4
- gt_v3_heading-5
- gt_v3_heading-6
- gt_v3_subheading
- gt_v3_paragraph
- gt_v3_caption

>[!note]
> - Công thức tinh size theo: Type scale 1.200 - Minor Third => https://type-scale.com/

>- Line height cua các thanh phan con lai = global line height
