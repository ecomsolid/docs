#### Margin
>[!note]
>Cách gọi tên: className = **.gt_v3_{m|mt|mr|mb|ml|mx|my}--{size}}**
>Ex: gt_v3_mt--xxs, gt_v3_m--xxs...

Chú thích:
- m : margin
- mt : margin top
- mb : margin bottom
- ml : margin left
- mr : margin right
- mx : margin left + margin right
- my : margin top + margin bottom

>[!warning]
>Support Responsive {lg|md|sm|xs}:{className}
>Ex: md:gt_v3_mt--xxs
#### Padding
>[!note]
>Cách gọi tên: className = **.gt_v3_{p|pt|pr|pb|pl|px|py}--{size}}**
>Ex: gt_v3_mt--xxs, gt_v3_m--xxs...

Chú thích:
- p : padding
- pt : padding top
- pb : padding bottom
- pl : padding left
- pr : padding right
- px : padding left + padding right
- py : padding top + padding bottom

>[!warning]
>Support Responsive {lg|md|sm|xs}:{className}
>Ex: md:gt_v3_pt--xxs

### **Typography**
>[!note]
>Cách gọi tên: className = **.gt_v3_{typoName}**
Danh sách class:
- gt_v3_heading-1
- gt_v3_heading-2
- gt_v3_heading-3
- gt_v3_heading-4
- gt_v3_heading-5
- gt_v3_heading-6

- gt_v3_caption
